Optimization Assembly Library - OpAsmLib - oal
Бибилиотека оптимизированного ассемблера
-----------------------------------------------
Эта библиотека содержит разные функции:
1) для работы с различными данными (num,float,str,array)
2) оптимизированные алгоритмы
3) низкоуровневые алгоритмы

Разбор файлов:
1) readme.txt - описание, информация
-------------------------------------
2) opasmlib_num_stdcall.asm - функции в формате stdcall для работы с целыми числами, пока только для высокоуровневых языков
3) opasmlib_float_stdcall.asm - функции в формате stdcall для работы с вещественными числами, пока только для высокоуровневых языков
4) opasmlib_str_stdcall.asm - функции в формате stdcall для работы со строками, пока только для высокоуровневых языков
5) opasmlib_array_stdcall.asm - функции в формате stdcall для работы с массивами, пока только для высокоуровневых языков
--------------------------------------------------------------------------------------------------------------------------------------

Во всех файлах есть подробные описания функций.
